# Guia Administrativo

Use `/admin/login` para entrar. O seed cria `admin@noitegamer.local` com senha de desenvolvimento documentada no README.

Rotinas:

- Dashboard: acompanha inscricoes, receita e vagas.
- Inscricoes: pesquisa visual e exportacao CSV.
- Pagamentos: status, gateway, id externo e webhooks.
- Check-in: informe o protocolo de inscricao confirmada.
- Torneios: gere a chave por modalidade e registre vencedores.
- Configuracoes: consulte dados do evento e modalidades.
